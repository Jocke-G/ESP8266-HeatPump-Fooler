# ESP8266 Heat Pump Fooler

Simulates the outdoor sensor, to make the heat pump work more when electricity is cheaper.

## Bill of Material

Wemos D1 Mini (ESP8266)
NTC 150 Thermistor
MCP4151-502E/P Digital Potentiometer
